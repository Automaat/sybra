## Description
Refactor the logging system to use structured logging with slog.

## Plan

### 1. Replace log.Printf with slog
- Convert all `log.Printf` calls to `slog.Info/Error/Debug`
- Add context fields where appropriate

### 2. Add log levels configuration
- Support `LOG_LEVEL` env var
- Default to `info` in production, `debug` in development

### 3. Add request ID tracing
- Generate UUID per request
- Pass through context
